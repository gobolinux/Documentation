title: "SystemInfo"

```
NAME
       SystemInfo -

SYNOPSIS
       SystemInfo

DESCRIPTION
       Display some basic system information. Useful for /etc/issue.

OPTIONS
       --terse

              Enable terse messages.

       --debug

              Enable debug messages.

       -h, --help

              Show this help.

       --version

              Show program version.

       -v, --verbose

              Enable verbose mode.

       --logfile <entry>

              Log all output to specified file.

GoboLinux                                                March 2017                                            SYSTEMINFO(1)
```
