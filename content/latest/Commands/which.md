title: "which"

```
NAME
       which -

SYNOPSIS
       which <executable>

DESCRIPTION
       Display real location of an executable file.

OPTIONS
       --terse

              Enable terse messages.

       --debug

              Enable debug messages.

       -h, --help

              Show this help.

       --version

              Show program version.

       -v, --verbose

              Enable verbose mode.

       --logfile <entry>

              Log all output to specified file.

EXAMPLES
              which ls
```
