---
title: "KeyManager"
---

Manage digital signature keys for GoboLinux packages.
```
Usage: KeyManager [ <key_file> ]

Options: 

-h, --help         Shows this help. 

-v, --version      Show program version. 

-V, --verbose      Enable verbose mode. 

-i, --import       Import key 

-l, --list         List public keys 
```
