```
NAME
       FixInfo -

SYNOPSIS
       FixInfo

DESCRIPTION
       Remakes the entries in the info 'dir' file.

OPTIONS
       --terse

              Enable terse messages.

       --debug

              Enable debug messages.

       -h, --help

              Show this help.

       --version

              Show program version.

       -v, --verbose

              Enable verbose mode.

       --logfile <entry>

              Log all output to specified file.

       -m, --mode <entry>

              Operation mode: 'links', 'dir' or 'both'.  The default value is 'both'.
```
